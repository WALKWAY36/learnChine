# StudyChine

### How to run project?
1) Create a virtual environment:
```
pip install virtualenv

virtualenv env
```

2) Activate env:
```
env\Scripts\activate
```

3) Install the requirements:
```
pip install -r requirements.txt
```

4) Run the App:
```
python manage.py runserver
```

5) Server started at http://127.0.0.1:8000/
